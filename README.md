# rollup-react-component-demo-2022

Rollup 编译的 react 组件工程

## Features

- [x] react jsx
- [x] postcss
- [x] eslint
- [x] copy typescript definition file
- [ ] typescript code

## Start

```sh
# eg. preparations
npm install

# eg. start the project
npm start

# eg. build the project
npm run build

# eg. test
npm test

# eg. release
npm run release
```
